Na Sua Estante

Plataforma web para organizar leituras, escrever resenhas e descobrir livros a partir de perguntas ao acervo, com frontend em Vue e backend em TypeScript usando Prisma + PostgreSQL; inclui scraping controlado do site Books to Scrape para popular o catálogo inicial.

Stack técnica

    Backend: Node.js + TypeScript, camadas separadas (domínio/aplicação/infra) e validação previsível.

    ORM: Prisma ORM com PostgreSQL, migrations versionadas e seeds reprodutíveis.

    Banco de dados: PostgreSQL via Docker Compose.

    Frontend: Vue para UI responsiva e focada no leitor.

    Scraping: Puppeteer (Books to Scrape) para ingestão inicial controlada.
